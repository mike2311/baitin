# Task 03-01: Demo Script

## Task Information

- **Phase**: 4 - Integration and Testing
- **Sprint**: Week 12
- **Priority**: High
- **Estimated Effort**: 1 day
- **Dependencies**: All modules complete and tested

## Objective

Create comprehensive demo script for stakeholder demonstration.

## Demo Script Structure

### 1. Introduction (2 minutes)
- PoC objectives recap
- What will be demonstrated
- Technology stack overview

### 2. Master Data Entry (5 minutes)
- **Item Entry:**
  - Create new item
  - Demonstrate keyboard navigation
  - Show lookup/search
  - Edit existing item
  - Show validation

- **Customer Entry:**
  - Create customer
  - Show fast data entry

### 3. Order Enquiry Entry (10 minutes)
- **OE Control:**
  - Create OE Control
  - Search OE Control

- **OE Entry:**
  - Create new OE
  - Add items using grid
  - Demonstrate item lookup
  - Show auto-calculate
  - Show auto-save
  - Demonstrate keyboard navigation

### 4. Excel Import (5 minutes)
- **Import:**
  - Upload Excel file
  - Show field mapping (auto-detect)
  - Execute import
  - Show error reporting
  - Verify imported data

### 5. Enquiry (3 minutes)
- **OE List:**
  - Search/filter OEs
  - View OE details
  - Show fast navigation

### 6. Q&A and Summary (5 minutes)
- Answer questions
- Recap key achievements
- Next steps (MVP)
